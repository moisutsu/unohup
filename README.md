> A useful wrapper library for nohup

## Install

```sh
cargo install --path .
```

## Usage

```sh
unohup <commands>
```

| Option         | description             |
| -------------- | ----------------------- |
| -l, --log-file | Specify a log file.     |
| -n, --nice     | Specify a `nice` value. |

The log is written to `/dev/null` by default, and the default value of `nice` is `0`.
